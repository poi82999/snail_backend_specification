## 0. 한 줄 비전

> **"무신사 SNAP × 캐치테이블"의 네일 특화 버전.**
> - 유저: **스타일 영감(스네일)** → **마음에 드는 디자인** → **샵 검색 → 예약**
> - 사장님: **디자인 업로드 → AI가 자동 분류 → 검색 노출 → 예약 받기**

## 3. 🎬 데모데이 시나리오 (12분 가정)

다음 **3개 골든 패스**가 백엔드 완성 시 시연 가능합니다.

### 🥇 골든 패스 1: 유저 발견 → 예약 → 리뷰 (8분)

```
1. 유저가 앱 실행 → 홈에서 인기 디자인 보기
2. 검색창에 "여리여리한 핑크" 입력 → ai_tags 매칭 결과
3. 디자인 상세 → 가격/디자이너/리뷰 확인 → 찜 ❤️
4. 예약하기 → 내일 14시 슬롯 선택 → 디자이너 자동 배정
5. (사장님 웹 전환) 알림톡 받음 + 캘린더에 신규 예약 표시
6. 사장님 수락 → 유저 앱에 푸시 도착
7. (시간 이동) 시술 완료 → 사장님이 완료 처리
8. 유저가 리뷰 5점 + 본문 + 사진 1장 작성
9. 사장님이 답글 작성
```

### 🥈 골든 패스 2: 사장님 디자인 등록 → AI 분석 (2분)

```
1. 사장님이 디자인 이미지 업로드
2. "분석 중" 상태 표시 (5~30초)
3. AI가 ai_tags + 색상 + 분위기 추출 완료
4. 디자인 카드가 앱 검색 결과에 즉시 노출
```

### 🥉 골든 패스 3: 커뮤니티 (2분)

```
1. 유저가 본인 네일 사진을 스네일에 게시
2. 본인의 완료 예약 연결 → 인증 뱃지 ✅
3. 다른 유저가 좋아요 + 댓글
4. 디자인 상세에 "이 디자인 받은 스네일" 섹션에 노출
```

## 6. 무신사 vs 캐치테이블 vs Snail

| 카테고리 | 무신사 | 캐치테이블 | **Snail** |
|---|---|---|---|
| 콘텐츠 발견 | SNAP, 셀럽, 큐레이션 | 매장 추천 | 스네일 + AI 태그 검색 |
| 검색 | 자연어 + 필터 | 위치/카테고리 | 자연어(AI 태그) + 필터 |
| 예약 / 구매 | 결제 → 배송 | 시간 선택 → 예약 → 알림톡 | **시간 선택 → 예약 → 알림톡 (결제 없음 or 계좌안내)** |
| 사용자 콘텐츠 | SNAP (사진/태그/좋아요) | 리뷰 | **스네일 + 리뷰 분리** |
| 커뮤니티 | 댓글/팔로우/랭킹 | 리뷰만 | 댓글/팔로우/랭킹 + **인증 뱃지** |
| 사장님 도구 | 셀러 콘솔 | 매장 캘린더 + 알림톡 | **샵 콘솔 + 캘린더 + AI 자동분석 + 알림톡** |
| 차별점 | — | — | **AI 디자인 자동 분류 + 스타일 커뮤니티** |
